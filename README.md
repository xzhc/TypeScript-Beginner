# TypeScript 学习项目

这是一个用于学习 TypeScript 的项目仓库。

## 项目结构

```
TypeScript-Beginner/
├── TypeScript-Learn/       # TypeScript学习环境
│   ├── src/                # 源代码目录
│   ├── dist/               # 编译输出目录
│   ├── note/               # 学习笔记
│   │   └── TypeScript环境搭建.md  # TypeScript环境搭建笔记
│   ├── package.json        # 项目配置
│   └── tsconfig.json       # TypeScript配置
└── README.md               # 主项目说明
```

## TypeScript-Learn

这是一个基础的 TypeScript 学习环境，包含以下内容：

- 完整的 TypeScript 项目结构
- 基础 TypeScript 示例代码
- 详细的环境搭建笔记

### 如何使用

1. 进入 TypeScript-Learn 目录：

   ```
   cd TypeScript-Learn
   ```

2. 安装依赖：

   ```
   npm install
   ```

3. 编译 TypeScript 代码：

   ```
   npm run build
   ```

4. 运行编译后的代码：

   ```
   npm run start
   ```

5. 开发模式（自动编译）：
   ```
   npm run dev
   ```
